## 目录结构
```
├── build # 编译配置及脚本
├── config # 配置
├── examples # demo 项目
├── components # 组件源码
├── src # 公共模块源码
├── test # 测试相关配置及结果
├── wiki # 文档
├── components.json # 组件定义
└── package.json
```
## 单元测试
```
npm run unit:watch
```

## 本地开发
先 `fork` 然后 `git clone`
```
npm init
npm start
```
之后访问 [http://localhost:8081](http://localhost:8081)

## 语法检查
```
npm run lint
```
